# betablocker

Chrome extension that blocks distracting sites on a schedule.

## Available npm commands

```
npm run format   # use prettier to format everything
npm run lint     # use eslint to check for issues
npm run build    # build to dist/ and remind the user to reload the extension
npm run prepare  # format → lint → build
```

## After making changes

Run `npm run prepare` so the extension is updated.
